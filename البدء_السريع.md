# 🚀 البدء السريع - نظام الترجمة

## ⚡ 3 خطوات فقط للبدء!

### 1️⃣ تأكد من الملفات
```
✅ /static/js/auto-translate.js
✅ /static/css/translation.css
✅ /templates/base.html (محدّث)
```

### 2️⃣ افتح الموقع
```bash
cd angry_store
python manage.py runserver
```

### 3️⃣ جرّب الترجمة!
- افتح المتصفح: http://localhost:8000
- اضغط على زر اللغة (🇬🇧 EN)
- شاهد الترجمة الفورية! ✨

## 🚀 كيفية الاستخدام:

### 1. تشغيل السيرفر:
```bash
cd angry_store
python manage.py runserver
```

### 2. الدخول للوحة التحكم:
```
http://127.0.0.1:8000/dashboard/
```

### 3. الصفحات المتاحة:
- **الإحصائيات:** `/dashboard/`
- **المنتجات:** `/dashboard/products/`
- **الطلبات:** `/dashboard/orders/`
- **المستخدمين:** `/dashboard/users/`
- **آراء العملاء:** `/dashboard/testimonials/`

## 🎨 التخصيص:

### تغيير الألوان:
افتح `static/css/dashboard-new.css` وعدل المتغيرات:
```css
/* الألوان موجودة في style.css الرئيسي */
--primary: #ff4757;
--secondary: #ffa502;
--card: #1e1e2e;
--dark: #16161e;
```

### تغيير الأيقونات:
الأيقونات موجودة في ملفات HTML، يمكنك تغييرها بسهولة:
```html
<div class="stat-icon">📦</div>  ← غير الإيموجي هنا
```

### إضافة صفحة جديدة:
1. أنشئ ملف HTML جديد في `templates/dashboard/`
2. استخدم `{% extends 'dashboard/base_dashboard.html' %}`
3. اكتب المحتوى في `{% block dashboard_content %}`

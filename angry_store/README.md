# 🎮 Angry Store - متجر الألعاب والهاردوير

متجر إلكتروني احترافي لبيع الألعاب الرقمية والاشتراكات والهاردوير

## 🚀 المميزات

- ✅ نظام تسجيل مستخدمين كامل
- ✅ لوحة تحكم احترافية للأدمن
- ✅ إدارة المنتجات (إضافة - تعديل - حذف)
- ✅ نظام طلبات متكامل
- ✅ طرق دفع متعددة (فودافون كاش - InstaPay - Binance Pay)
- ✅ رفع صور إثبات الدفع
- ✅ تصميم متجاوب (Responsive)
- ✅ نظام بحث وفلترة
- ✅ حماية CSRF
- ✅ تشفير كلمات المرور

## 📋 المتطلبات

- Python 3.8+
- Django 4.2.7
- Pillow 10.1.0

## 🔧 التثبيت والتشغيل

### 1. إنشاء بيئة افتراضية

```bash
python -m venv venv
venv\Scripts\activate
```

### 2. تثبيت المتطلبات

```bash
pip install -r requirements.txt
```

### 3. إنشاء قاعدة البيانات

```bash
python manage.py makemigrations
python manage.py migrate
```

### 4. إنشاء حساب أدمن

```bash
python manage.py createsuperuser
```

أدخل:
- اسم المستخدم
- البريد الإلكتروني
- كلمة المرور

### 5. تشغيل السيرفر

```bash
python manage.py runserver
```

### 6. فتح الموقع

افتح المتصفح على: `http://127.0.0.1:8000`

## 📱 الصفحات الرئيسية

- **الصفحة الرئيسية**: `/`
- **تسجيل دخول**: `/login/`
- **إنشاء حساب**: `/register/`
- **لوحة التحكم**: `/dashboard/` (للأدمن فقط)
- **إدارة المنتجات**: `/dashboard/products/`
- **إدارة المستخدمين**: `/dashboard/users/`
- **إدارة الطلبات**: `/dashboard/orders/`

## 📊 قاعدة البيانات

### النماذج (Models):

1. **UserProfile**: معلومات المستخدمين
   - الاسم
   - البريد الإلكتروني
   - رقم الهاتف
   - تاريخ التسجيل

2. **Product**: المنتجات
   - الاسم
   - الوصف
   - السعر
   - النوع (لعبة/اشتراك/هاردوير)
   - الصورة

3. **Order**: الطلبات
   - المستخدم
   - المنتج
   - طريقة الدفع
   - Discord Username
   - رقم WhatsApp
   - صورة إثبات الدفع
   - الحالة (قيد المراجعة/مكتمل/مرفوض)

## 🛠️ التخصيص

### تغيير أرقام الدفع:

في ملف `store/views.py` في دالة `create_order`:

```python
payment_numbers = {
    'vodafone': '01234567890',  # غير هذا الرقم
    'instapay': 'angrystore@instapay',  # غير هذا
    'binance': 'angrystore@binance.com'  # غير هذا
}
```

### تغيير الألوان:

في ملف `static/css/style.css`:

```css
:root {
    --primary: #e63946;  /* اللون الأحمر */
    --secondary: #8b5cf6;  /* اللون البنفسجي */
    --dark: #0a0a0a;  /* الخلفية */
}
```

## 📦 التحويل إلى PostgreSQL

في ملف `config/settings.py`:

```python
DATABASES = {
    'default': {
        'ENGINE': 'django.db.backends.postgresql',
        'NAME': 'angry_store',
        'USER': 'your_username',
        'PASSWORD': 'your_password',
        'HOST': 'localhost',
        'PORT': '5432',
    }
}
```

ثم:
```bash
pip install psycopg2
python manage.py migrate
```

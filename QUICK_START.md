## 🎨 التخصيص

### تغيير الألوان:
افتح `static/css/style.css` وعدّل:
```css
:root {
    --primary: #e63946;  /* اللون الأساسي */
    --accent: #f59e0b;   /* اللون الثانوي */
}
```

### إضافة قسم جديد:
1. عدّل `store/models.py`
2. عدّل `templates/base.html` (Navbar)
3. عدّل `templates/home.html` (Categories)

## 🎯 نصائح للاستخدام

### للصور:
- استخدم صور عالية الجودة
- النسبة المثالية: 16:10
- الحجم الموصى به: 1920x1200px
- الصيغة: JPG أو PNG

### للمنتجات:
- عنوان واضح ومختصر
- وصف جذاب (2-3 أسطر)
- سعر واضح
- اختر النوع الصحيح

### للأداء:
- ضغط الصور قبل الرفع
- استخدم WebP إن أمكن
- تجنب الصور الكبيرة جداً

## 🆘 حل المشاكل

### الصور لا تظهر:
```bash
python manage.py collectstatic
```

### التصميم لا يعمل:
1. امسح الكاش (Ctrl+Shift+R)
2. تأكد من مسار CSS صحيح
3. تحقق من Console للأخطاء

### الخط لا يظهر:
- تأكد من اتصال الإنترنت
- Google Fonts يحتاج إنترنت
- أو حمّل الخط محلياً

## 🚀 كيف يعمل النظام الآن؟

### 1. ترجمة الجمل الكاملة (وليس كلمة كلمة)
```javascript
// في translate.js
'احصل على منتجك فوراً بعد الدفع مباشرة': 'Get your product instantly after payment',
'مفاتيح التفعيل': 'Activation Keys',
'توصيل فوري •   • دعم 24/7': 'Instant Delivery • Full Warranty • 24/7 Support',
```

### 2. ترجمة ثنائية الاتجاه تلقائياً
- تكتب الترجمة مرة واحدة (عربي → إنجليزي)
- النظام يعكسها تلقائياً (إنجليزي → عربي)

### 3. حفظ اللغة المختارة
- يحفظ في localStorage
- عند العودة للموقع، اللغة تبقى كما اخترتها

## 🎨 كيف تستخدم النظام؟

### في أي صفحة HTML:

#### 1. تأكد من تحميل السكريبت (موجود في base.html):
```html
<script src="/static/js/translate.js"></script>
```

#### 2. زر تبديل اللغة (موجود في base.html):
```html
<button onclick="toggleLanguage()">
    <span id="langIcon">🇬🇧</span>
    <span id="langText">EN</span>
</button>
```

#### 3. اكتب المحتوى بالعربية عادي:
```html
<h1>احصل على منتجك فوراً بعد الدفع مباشرة</h1>
<p>مفاتيح التفعيل</p>
<button>توصيل فوري •   • دعم 24/7</button>
```

#### 4. عند الضغط على زر EN:
```html
<h1>Get your product instantly after payment</h1>
<p>Activation Keys</p>
<button>Instant Delivery • Full Warranty • 24/7 Support</button>
```

## ➕ كيف تضيف ترجمة جديدة؟

### خطوة واحدة فقط:

1. افتح `angry_store/static/js/translate.js`
2. ابحث عن `const translations = {`
3. أضف جملتك:

```javascript
const translations = {
    // الترجمات الموجودة...
    
    // أضف ترجمتك الجديدة هنا
    'النص العربي الكامل': 'Full English Translation',
};
```

4. احفظ الملف
5. حدّث الصفحة في المتصفح
6. تم! ✨

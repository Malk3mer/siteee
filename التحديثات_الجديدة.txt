# 🎉 التحديثات الجديدة - Angry Store

## ✅ تم إصلاح وإضافة:

### 1. ✅ إصلاح مشكلة التاجات
**المشكلة:** لم تكن التاجات تظهر للاختيار
**الحل:** تم إعادة كتابة الكود بطريقة صحيحة
- الآن يمكنك اختيار التاجات من checkboxes
- كل تاج له checkbox منفصل
- التاجات المختارة تظهر بلون مميز
- يجب اختيار من 2 إلى 5 تاجات

### 2. 🔐 تغيير كلمة المرور وإرسالها بالإيميل
**الميزة الجديدة:**
- يمكن للأدمن تغيير كلمة مرور أي مستخدم
- يتم إرسال كلمة المرور الجديدة تلقائياً للإيميل
- إذا لم يكن للمستخدم إيميل، تظهر كلمة المرور في رسالة

**كيفية الاستخدام:**
1. اذهب إلى Dashboard → المستخدمين
2. اضغط على "🔑 تغيير كلمة المرور" بجانب المستخدم
3. أدخل كلمة المرور الجديدة وأكدها
4. سيتم إرسال الإيميل تلقائياً

### 3. 💬 رابط Discord في الموقع
**تم إضافة رابط Discord في:**
- الـ Navbar (أعلى الصفحة)
- الـ Footer (أسفل الصفحة)
- تصميم جميل مع أيقونة Discord
- الرابط: https://discord.gg/MvBgwsyzKP

## 📋 إعداد الإيميل

لكي تعمل ميزة إرسال كلمة المرور بالإيميل، يجب إعداد Gmail:

### الخطوة 1: إنشاء App Password في Gmail
1. اذهب إلى حسابك في Google
2. اذهب إلى Security → 2-Step Verification
3. في الأسفل، اضغط على "App passwords"
4. اختر "Mail" و "Other (Custom name)"
5. اكتب "Angry Store"
6. انسخ الـ App Password (16 حرف)

### الخطوة 2: تحديث settings.py
افتح ملف `config/settings.py` وغيّر:

```python
EMAIL_HOST_USER = 'your-email@gmail.com'  # ضع إيميلك هنا
EMAIL_HOST_PASSWORD = 'xxxx xxxx xxxx xxxx'  # ضع App Password هنا
DEFAULT_FROM_EMAIL = 'Angry Store <your-email@gmail.com>'
```

### الخطوة 3: اختبار الإيميل
```bash
python manage.py shell
```

ثم:
```python
from django.core.mail import send_mail
send_mail(
    'Test',
    'This is a test email',
    'your-email@gmail.com',
    ['recipient@example.com'],
)
```

## 🚀 كيفية الاستخدام

### تشغيل الموقع:
```bash
cd angry_store
python manage.py runserver
```

### إضافة منتج جديد:
1. اذهب إلى: http://127.0.0.1:8000/dashboard/products/add/
2. املأ البيانات
3. اختر من 2 إلى 5 تاجات (الآن يعمل!)
4. ارفع صورة أو ضع رابط
5. احفظ المنتج

### تغيير كلمة مرور مستخدم:
1. اذهب إلى: http://127.0.0.1:8000/dashboard/users/
2. اضغط "🔑 تغيير كلمة المرور"
3. أدخل كلمة المرور الجديدة
4. سيتم إرسال إيميل تلقائياً

### الانضمام لـ Discord:
- اضغط على زر Discord في الـ Navbar أو Footer
- أو اذهب مباشرة: https://discord.gg/MvBgwsyzKP

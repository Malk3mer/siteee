# 🚀 Quick Start - Translation System

## ⚡ Just 3 Steps to Get Started!

### 1️⃣ Verify Files
```
✅ /static/js/auto-translate.js
✅ /static/css/translation.css
✅ /templates/base.html (updated)
```

### 2️⃣ Start the Server
```bash
cd angry_store
python manage.py runserver
```

### 3️⃣ Try Translation!
- Open browser: http://localhost:8000
- Click language button (🇬🇧 EN)
- Watch instant translation! ✨

### 2. نظام الترجمة الجديد:
- كل نص عربي تم إضافة `data-ar` و `data-en` له
- اللغة الافتراضية: **الإنجليزية**
- التبديل بين اللغات بضغطة زر واحدة
- حفظ اللغة المختارة تلقائياً

### 3. كيفية عمل النظام:

```html
<!-- مثال بسيط -->
<span data-ar="اشتري الآن" data-en="Buy Now">Buy Now</span>

<!-- عند الضغط على زر اللغة -->
<!-- إذا اخترت العربية → يظهر: اشتري الآن -->
<!-- إذا اخترت الإنجليزية → يظهر: Buy Now -->
```

### 4. زر تغيير اللغة:
- موجود في الـ Navbar أعلى الصفحة
- 🇬🇧 EN → للتبديل للإنجليزية
- 🇦🇪 AR → للتبديل للعربية

### 5. المميزات:
✨ تبديل فوري بدون إعادة تحميل الصفحة
💾 حفظ اللغة المختارة في المتصفح
🔄 تحديث تلقائي لجميع النصوص
📱 دعم كامل للـ RTL (عربي) و LTR (إنجليزي)
🎯 سهل الاستخدام والتعديل

### 7. كيفية إضافة ترجمة لصفحة جديدة:

1. افتح الملف
2. ابحث عن أي نص عربي
3. غيره من:
   ```html
   <span>النص العربي</span>
   ```
   إلى:
   ```html
   <span data-ar="النص العربي" data-en="English Text">English Text</span>
   ```

### 8. أمثلة:

#### للأزرار:
```html
<button><span data-ar="إضافة للسلة" data-en="Add to Cart">Add to Cart</span></button>
```

#### للعناوين:
```html
<h1><span data-ar="الصفحة الرئيسية" data-en="Home Page">Home Page</span></h1>
```

#### للـ Input Placeholder:
```html
<input type="text" 
       placeholder="Search..." 
       data-ar-placeholder="ابحث..." 
       data-en-placeholder="Search...">
```

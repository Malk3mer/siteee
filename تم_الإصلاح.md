```bash
cd angry_store
python manage.py runserver
```

ثم افتح المتصفح على: http://127.0.0.1:8000/

## ⚠️ ملاحظات مهمة:

1. **إذا لم تظهر التغييرات:**
   - اضغط `Ctrl + Shift + R` لتحديث الصفحة
   - أو امسح cache المتصفح

2. **إذا ظهرت مشاكل في الصور:**
   - تأكد من وجود الصور في مجلد `media/`
   - تأكد من رفع الصور من لوحة التحكم

3. **للإنتاج (Production):**
   - غير `DEBUG = False` في `settings.py`
   - شغل `python manage.py collectstatic`
   - غير `SECRET_KEY` لمفتاح جديد

## 🔧 أوامر مفيدة:

```bash
# تشغيل السيرفر
python manage.py runserver

# إنشاء admin
python manage.py createsuperuser

# جمع الملفات الثابتة
python manage.py collectstatic

# عمل migrations
python manage.py makemigrations
python manage.py migrate

# التحقق من الأخطاء
python manage.py check
```

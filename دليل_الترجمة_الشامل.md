## 📋 كيفية إضافة ترجمات جديدة

### الخطوة 1: افتح ملف الترجمة
```
angry_store/static/js/translate.js
```

### الخطوة 2: أضف الترجمة في قسم translations
```javascript
const translations = {
    // أضف الجملة العربية والترجمة الإنجليزية
    'النص العربي الكامل': 'Full English Translation',
    
    // مثال:
    'مرحباً بك في المتجر': 'Welcome to the Store',
    'اشترك الآن واحصل على خصم': 'Subscribe now and get a discount',
};
```

### ⚠️ ملاحظات مهمة:
1. **اكتب الجملة كاملة** - لا تقسمها لكلمات منفصلة
2. **استخدم علامات الترقيم بدقة** - النقاط والفواصل مهمة
3. **الجمل الأطول لها أولوية** - النظام يترجم الجمل الطويلة أولاً

## 🔧 كيفية استخدام النظام

### في ملفات HTML:
```html
<!-- 1. تأكد من تحميل السكريبت -->
<script src="/static/js/translate.js"></script>

<!-- 2. أضف زر تبديل اللغة -->
<button onclick="toggleLanguage()">
    <span id="langIcon">🇬🇧</span>
    <span id="langText">EN</span>
</button>

<!-- 3. اكتب النصوص بالعربية عادي -->
<h1>مرحباً بك في المتجر</h1>
<p>احصل على منتجك فوراً بعد الدفع مباشرة</p>
```

### منع ترجمة عنصر معين:
```html
<!-- استخدم data-no-translate -->
<div data-no-translate>
    هذا النص لن يترجم
</div>
```

## 🎨 تطبيق الترجمة على صفحات جديدة

### 1. تأكد من وجود السكريبت في base.html:
```html
<script src="/static/js/translate.js"></script>
```

### 2. أضف زر تبديل اللغة في الـ navbar:
```html
<button onclick="toggleLanguage()">
    <span id="langIcon">🇬🇧</span>
    <span id="langText">EN</span>
</button>
```

### 3. اكتب المحتوى بالعربية:
- النظام سيترجم تلقائياً عند الضغط على زر التبديل
- الترجمة تحفظ في localStorage
- عند العودة للموقع، اللغة المختارة تبقى محفوظة

---

## 🔄 ترجمة العملة

النظام يترجم العملة تلقائياً:
- `299 ج` → `299 EGP`
- `1500 جنيه` → `1500 EGP`
- `450 ج.م` → `450 EGP`

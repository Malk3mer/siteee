# 🚀 دليل سريع: كيف تضيف ترجمة جديدة

## الطريقة الصحيحة ✅

### 1. افتح ملف الترجمة
```
angry_store/static/js/translate.js
```

### 2. ابحث عن const translations
```javascript
const translations = {
    // هنا تضيف الترجمات
```

### 3. أضف الجملة كاملة
```javascript
const translations = {
    // الجمل الموجودة...
    
    // أضف جملتك الجديدة هنا:
    'النص العربي الكامل': 'Full English Text',
};
```

---

## ❌ أخطاء شائعة

### خطأ 1: تقسيم الجملة
```javascript
// ❌ خطأ
'احصل': 'Get',
'على': 'on',
'منتجك': 'your product',

// ✅ صح
'احصل على منتجك فوراً': 'Get your product instantly',
```

### خطأ 2: نسيان علامات الترقيم
```javascript
// ❌ خطأ
'توصيل فوري ضمان دعم': 'Instant Delivery Warranty Support',

// ✅ صح
'توصيل فوري • ضمان • دعم 24/7': 'Instant Delivery • Warranty • 24/7 Support',
```

## 📝 أمثلة عملية

### مثال 1: ترجمة عنوان

```javascript
'نوفر لك أفضل الألعاب بأقل الأسعار': 'We provide you the best games at the lowest prices',
```

### مثال 3: ترجمة زر
```javascript
'اشترك الآن واحصل على خصم 50%': 'Subscribe now and get 50% discount',
```

---

## 🎯 خطوات سريعة

1. **افتح** `translate.js`
2. **ابحث** عن `const translations = {`
3. **أضف** جملتك: `'النص العربي': 'English Text',`
4. **احفظ** الملف
5. **حدّث** الصفحة في المتصفح
6. **اضغط** زر تبديل اللغة
7. **شاهد** الترجمة تعمل! ✨

---

## 💡 نصيحة ذهبية

**اكتب الجملة كاملة كما هي في HTML**

إذا كان في HTML:
```html
<h1>احصل على منتجك فوراً بعد الدفع مباشرة</h1>
```

في translate.js:
```javascript
'احصل على منتجك فوراً بعد الدفع مباشرة': 'Get your product instantly after payment',
```

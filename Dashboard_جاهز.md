## 🚀 كيف تستخدم Dashboard:

### 1. شغل السيرفر:
```bash
cd angry_store
python manage.py runserver
```

### 2. افتح المتصفح:
```
http://127.0.0.1:8000/dashboard/
```

### 3. الصفحات:
- **الإحصائيات:** `/dashboard/`
- **المنتجات:** `/dashboard/products/`
- **الطلبات:** `/dashboard/orders/`
- **المستخدمين:** `/dashboard/users/`
- **آراء العملاء:** `/dashboard/testimonials/`

## ⚠️ ملاحظات:

1. **لازم تكون Admin:**
   - Dashboard يظهر فقط للـ Admin
   - سجل دخول بحساب Admin

2. **إذا ما ظهر التصميم:**
   - اضغط Ctrl+Shift+R
   - أو امسح Cache المتصفح

3. **الملفات القديمة:**
   - يمكنك حذف `dashboard.css` القديم
   - يمكنك حذف `orders.css` القديم
   - الآن كل شيء في `dashboard-new.css`

---

## 🔧 التعديل:

### لو عايز تغير الألوان:
افتح `static/css/dashboard-new.css`

### لو عايز تضيف صفحة جديدة:
1. أنشئ ملف HTML في `templates/dashboard/`
2. استخدم `{% extends 'dashboard/base_dashboard.html' %}`
3. اكتب المحتوى

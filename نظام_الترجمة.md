## 🚀 كيفية الاستخدام

### للمستخدم:
1. اضغط على زر اللغة في أعلى الصفحة (🇬🇧 EN / 🇦🇪 AR)
2. سيتم تبديل اللغة فوراً لجميع عناصر الصفحة
3. سيتم حفظ اختيارك تلقائياً

### للمطور:

#### إضافة ترجمة جديدة:
```javascript
// في ملف auto-translate.js
const translations = {
    'النص بالعربية': 'English Text',
    // أضف المزيد هنا
};
```

#### منع ترجمة عنصر معين:
```html
<div data-no-translate>
    هذا النص لن يترجم
</div>
```

#### إضافة class للعملة:
```html
<span class="currency">100 ج</span>
<!-- سيترجم تلقائياً إلى: 100 EGP -->
```

## 🎨 التخصيص

### تغيير أيقونات اللغة:
```javascript
// في auto-translate.js
if (langIcon) langIcon.textContent = '🇦🇪'; // للإنجليزية
if (langIcon) langIcon.textContent = '🇬🇧'; // للعربية
```

### تخصيص أنماط RTL/LTR:
```css
/* في translation.css */
html[lang="en"] {
    direction: ltr;
    /* أضف أنماطك هنا */
}

html[lang="ar"] {
    direction: rtl;
    /* أضف أنماطك هنا */
}
```

## 🔧 استكشاف الأخطاء

### المشكلة: الترجمة لا تعمل
**الحل:**
1. تأكد من تحميل ملف `auto-translate.js`
2. افتح Console في المتصفح وتحقق من الأخطاء
3. تأكد من وجود `id="langIcon"` و `id="langText"` في زر اللغة

### المشكلة: بعض النصوص لا تترجم
**الحل:**
1. أضف الترجمة في قاموس `translations`
2. تأكد من عدم وجود `data-no-translate` على العنصر
3. تحقق من أن النص مطابق تماماً في القاموس

### المشكلة: اتجاه النص غير صحيح
**الحل:**
1. تأكد من تحميل ملف `translation.css`
2. تحقق من أن `lang` و `dir` يتغيران في `<html>`

## 📁 الملفات المهمة:

```
angry_store/
├── templates/
│   ├── base.html                    ← القالب الأساسي
│   ├── home.html                    ← الصفحة الرئيسية
│   ├── cart.html                    ← السلة (محدث)
│   ├── 404.html                     ← صفحة الخطأ
│   └── dashboard/
│       ├── base_dashboard.html      ← قالب Dashboard
│       ├── dashboard.html           ← الإحصائيات
│       ├── products.html            ← المنتجات
│       ├── orders.html              ← الطلبات
│       └── users.html               ← المستخدمين
│
├── static/
│   ├── css/
│   │   ├── style.css                ← الستايل الرئيسي
│   │   ├── dashboard-new.css        ← Dashboard
│   │   └── enhancements.css         ← ميزات إضافية
│   └── js/
│       ├── script.js                ← السكريبت الرئيسي
│       ├── translate.js             ← الترجمة
│       └── enhancements.js          ← ميزات إضافية
│
└── store/
    ├── views.py                     ← الوظائف (محدث)
    ├── urls.py                      ← الروابط (محدث)
    └── models.py                    ← قاعدة البيانات
```

## 🚀 التشغيل:

```bash
cd angry_store
python manage.py runserver
```

**الرابط:** http://127.0.0.1:8000/

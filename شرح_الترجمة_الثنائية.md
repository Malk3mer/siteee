# 🔄 نظام الترجمة الثنائي الاتجاه

## كيف يعمل النظام؟

### 📖 القاموس الأساسي (عربي → إنجليزي)
```javascript
const translations = {
    'احصل على منتجك فوراً بعد الدفع مباشرة': 'Get your product instantly after payment',
    'مفاتيح التفعيل': 'Activation Keys',
    'توصيل فوري •   • دعم 24/7': 'Instant Delivery • Full Warranty • 24/7 Support',
};
```

### 🔄 القاموس العكسي (إنجليزي → عربي)
النظام يقوم **تلقائياً** بعكس القاموس:
```javascript
const reverseTranslations = {
    'Get your product instantly after payment': 'احصل على منتجك فوراً بعد الدفع مباشرة',
    'Activation Keys': 'مفاتيح التفعيل',
    'Instant Delivery • Full Warranty • 24/7 Support': 'توصيل فوري •   • دعم 24/7',
};
```

---

## ✨ المميزات

### 1. ترجمة ثنائية الاتجاه
- عربي → إنجليزي ✅
- إنجليزي → عربي ✅

### 2. ترجمة الجمل الكاملة
```
❌ خطأ: "احصل" + "على" + "منتجك" = "Get" + "on" + "your product"
✅ صح: "احصل على منتجك فوراً" = "Get your product instantly"
```

### 3. حفظ اللغة المختارة
- يحفظ في localStorage
- عند العودة للموقع، اللغة تبقى كما اخترتها

### 4. تغيير الاتجاه تلقائياً
- عربي: RTL (من اليمين لليسار)
- إنجليزي: LTR (من اليسار لليمين)

## 🎯 كيف تضيف ترجمة جديدة؟

### خطوة واحدة فقط!
افتح `translate.js` وأضف في القاموس:

```javascript
const translations = {
    // الترجمات الموجودة...
    
    // أضف ترجمتك الجديدة (عربي → إنجليزي فقط)
    'النص العربي الكامل': 'Full English Text',
};
```

**النظام سيقوم تلقائياً بإنشاء الترجمة العكسية!** 🎉

## 📝 أمثلة عملية

### مثال 1: صفحة رئيسية
```javascript
'اكتشف عالم الألعاب الرقمية': 'Discover the World of Digital Games',
```

**النتيجة:**
- عند الضغط على EN: "Discover the World of Digital Games"
- عند الضغط على AR: "اكتشف عالم الألعاب الرقمية"

### مثال 2: قائمة التنقل
```javascript
'ألعاب أوفلاين': 'Offline Games',
'ألعاب أونلاين': 'Online Games',
'مفاتيح التفعيل': 'Activation Keys',
```

**النتيجة:**
- EN: Offline Games, Online Games, Activation Keys
- AR: ألعاب أوفلاين, ألعاب أونلاين, مفاتيح التفعيل

### مثال 3: جمل طويلة
```javascript
'احصل على منتجك فوراً بعد الدفع مباشرة': 'Get your product instantly after payment',
```

**النتيجة:**
- EN: "Get your product instantly after payment"
- AR: "احصل على منتجك فوراً بعد الدفع مباشرة"

## ⚡ سريع وسهل

1. **أضف** الترجمة في القاموس (عربي → إنجليزي فقط)
2. **النظام** يعكسها تلقائياً (إنجليزي → عربي)
3. **اضغط** زر تبديل اللغة
4. **شاهد** السحر يحدث! ✨

## 💡 ملاحظة مهمة

**لا تحتاج لإضافة الترجمة العكسية يدوياً!**

```javascript
// ❌ لا تفعل هذا
const translations = {
    'مفاتيح التفعيل': 'Activation Keys',
};
const reverseTranslations = {
    'Activation Keys': 'مفاتيح التفعيل', // لا داعي لهذا!
};

};
```

## 🎯 كيفية تحديث الملفات المتبقية:

### الخطوات:

1. **افتح الملف**
2. **ابحث عن أي نص عربي**
3. **استبدله بهذا النمط:**

```html
<!-- قبل -->
<h1>النص العربي</h1>

<!-- بعد -->
<h1><span data-ar="النص العربي" data-en="English Text">English Text</span></h1>
```

### أمثلة سريعة:

#### للعناوين:
```html
<h1><span data-ar="ألعاب أونلاين" data-en="Online Games">Online Games</span></h1>
```

#### للأزرار:
```html
<button><span data-ar="حفظ" data-en="Save">Save</span></button>
```

#### للرسائل:
```html
<p><span data-ar="لا توجد بيانات" data-en="No data">No data</span></p>
```

#### للـ Placeholders:
```html
<input placeholder="Search..." 
       data-ar-placeholder="ابحث..." 
       data-en-placeholder="Search...">
```
